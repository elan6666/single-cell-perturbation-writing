# Drug Study Card

Complete this factual card before drafting a drug-perturbation paper. Use
`unknown` rather than guessing.

```text
Task and primary endpoint:

Drug exposure
- Canonical name / aliases / identifier:
- SMILES, InChIKey, or biologic identity:
- Salt, stereochemistry, formulation, vehicle:
- Single agent or combination; component ratio/order:
- Dose(s), unit(s), and range:
- Exposure time(s) and schedule:

Biological system
- Assay and readout:
- Cell model, tissue/disease context, cell state:
- Donors, replicates, plates/batches:
- Viability/cell-count and QC policy:

Model and evaluation
- Drug representation available to each model:
- Held-out unit and split group key:
- Baseline input parity:
- Primary and secondary metrics:

Claim ladder
- Directly measured finding:
- Supported mechanism hypothesis:
- Permitted application statement:
- Explicitly unsupported translation claim:
- Provenance locations:
```
