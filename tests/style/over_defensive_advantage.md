# Style Fixture: Over-Defensive Advantage

Bad input:

```text
TriShift achieves the highest Overlap@20 on Norman, but Adamson and Norman
differences are small and should be treated only as directionally consistent
evidence rather than a strong ranking conclusion.
```

Expected fix:

- state the advantage first;
- keep the strongest setting visible;
- consolidate routine caveats into a boundary sentence only if needed;
- avoid weakening a supported result immediately after presenting it.
